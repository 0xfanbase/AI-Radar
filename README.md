# AI Frontier Wire

A free, public, fully autonomous AI-news site: a fact-checked daily wire of
what happened in frontier AI and why it matters, a living tracker of
frontier models across US, Chinese, and open-weights labs (**the Frontier
Board**), and a cross-linked plain-English glossary (**the Lexicon**) so a
newcomer can read the site and actually understand the space.

The site curates, writes, fact-checks, and publishes itself on a daily
schedule, and audits its own accuracy in public. See `CLAUDE.md` for the
full editorial rules and architecture, `PROGRESS.md` for the build log, and
`IMPROVEMENT_BACKLOG.md` for every judgment call made along the way.

**Non-commercial · anonymous · fully auto-published.** This project is not
run for profit, publishes under a bot identity with no personal
identifiers, and every page it generates carries a disclaimer that content
is AI-curated and AI-written, with links out to primary sources. Code is
MIT-licensed; published editorial content (cards, Frontier Board, Lexicon)
is CC BY 4.0.

This repository is under active, incremental build — see `PROGRESS.md` for
current status.
